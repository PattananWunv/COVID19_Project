# COVID-19 Data Visualization and Analysis
This project explores global COVID-19 cases, deaths, and vaccination trends, allowing clear comparisons across different regions and time periods. Using SQL for data cleaning and metric calculations, and Tableau for building interactive dashboards.

## Tools 
- SQL Server Management Studio (SSMS) – data cleaning, transformation, and metric calculations
- Tableau – interactive dashboards and data visualization
- Excel / CSV – data source files
